# Succedo Desktop App

## Deployment

### Compile snap

```
snapcraft --use-lxd
```

Also see https://experimental-software.atlassian.net/wiki/spaces/~958839839/blog/2020/12/30/3080591/LXD+Reset+configuration

### Install snap

```
snap install --devmode ./succedo_0.1.2_amd64.snap
```
